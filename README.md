# ooplab

这是一个 Java 面向对象程序设计课程实验仓库。

仓库只保存代码和可以公开的实验材料，不上传带有个人信息的实验报告、提交压缩包、运行日志等私有文件。这些内容统一放在 `_private/` 中，并通过 `.gitignore` 忽略。

## 目录结构

- `lab1/`：Lab 1 代码、Maven 配置，以及公开保留的 `lab1.pdf`
- `lab2/`：后续实验代码与公开材料目录
- `_private/`：私有实验报告、提交压缩包、日志等，不纳入 Git
- `.gitignore`：忽略私有文件和构建产物

## 当前实验进度

### Lab 1

`lab1` 是一个基于 Java 17、Maven 和 Lanterna 的黑白棋初版程序。

目前已经实现：

- 终端界面显示棋盘
- 支持输入类似 `D4` 的坐标
- 合法输入后轮流落子
- 支持输入 `quit` 退出
- 在 Windows 下提供文本终端失败后的回退方案

目前还没有实现：

- 黑白棋初始四子布局
- 合法走子搜索
- 翻子规则
- 无子可下时跳过回合
- 胜负结算

## 运行方式

环境要求：

- JDK 17
- Maven

运行 `lab1`：

```bash
cd lab1
mvn clean compile exec:java
```

## 文件管理约定

以下内容保留在仓库中：

- 源代码
- `pom.xml` 等构建配置
- 可以公开的实验说明 PDF

以下内容放入 `_private/`，不上传到 GitHub：

- 带姓名、学号等个人信息的实验报告
- 提交用的 zip 压缩包
- 本地运行日志
- 其他不适合公开的材料

## 后续实验管理方式

后续实验按相同规则维护：

- 每个实验使用独立目录，如 `lab2/`、`lab3/`
- 代码和公开材料放在对应 `labX/` 目录
- 私有材料统一放在 `_private/`
